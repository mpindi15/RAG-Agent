Document Q&A · Gemini + ChromaDB

How this works

  1. Upload files in the Documents tab
  2. Text is chunked and embedded into ChromaDB
  3. Your question retrieves the top matching chunks
  4. Gemini answers using only those chunks, with citations

Every answer is logged with latency, token usage, and estimated cost — see the Monitoring and Traces tabs.
